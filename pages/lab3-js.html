TNMK30 Laboration 3: JavaScript

I den här laborationen ska vi undersöka hur vi kan koda in beteende för en webbsida med scriptspråket JavaScript.

I HTML5 finns ett nytt element som heter <canvas>, som kan användas för att skapa interaktiv grafik med hjälp av JavaScript. Vi har gjort i ordning några exempel som visar hur man kan rita några enkla former, men också hur man kan få något att röra på sig. Här är även ett exempel på hur man kan få en funktion att anropas när användaren klickar på en knapp.

Uppgift

För godkänt ska din webbapplikation:

  • Använda en loop för att rita upp flera objekt, eller rita en spännande form.
  • Innehålla en animation eller reagera på att användaren gör något (till exempel klickar på en knapp eller rör på musen).
  • Struktureras med hjälp av funktioner.

Redovisa för en labbassistent.

Idétorka?

Ni är fria att göra vad ni vill, men här är några exempel som skulle kunna vara av rimlig omfattning:

  • Ett program, där man får välja dimensionerna hos en legobit som sedan ritas upp
  • Några bollar som studsar runt
  • En gubbe som nickar eller skakar på huvudet
  • Snöflingor som faller
  • Ett enkelt ritprogram
  • Ett mönster som användaren får välja färg på

Var det för svårt?

Här finns en förberedelseuppgift att börja med för att bekanta er med JavaScript, innan ni löser labbuppgiften.

1. Länka in js-filen

Det första vi behöver göra är att, precis som med css, länka in vår js-fil i html-dokumentet. Vi vill oftast ladda in script sist. Så här kan det se ut: 

<!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8">
    <title>TNMK30 Lab 3</title>
</head>
<body>
    <!-- kanske lite html -->
    <script src="minjavascriptfil.js"></script>
</body>
</html>

2. Se till att ha konsollen öppen

Att koda JavaScript är ganska likt c++ och det första vi lärde oss i c++ var antagligen att skriva ut saker i kommandotolken med cout. De allra flesta webbläsare har en egen kommandotolk, eller console, som vi kan skriva till. Denna finns under tools (chrome) eller tools > web developer (firefox). Öppna den, så kan vi se vad som händer i vår JavaScript.

4. Ett program i C++

Nedan finns ett enkelt program skrivet i c++ som skriver ut en array. Översätt det till JavaScript. 

#include <iostream>
using namespace std;

void printArray(int array[], int size) {
  for (int i = 0; i < size; i++) {
    cout << array[i] << endl;
  }
}

int main() {
  int foo[] = {1, 3, 3, 7};
  printArray(foo, 4);
}
