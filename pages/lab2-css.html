TNMK30 Laboration 2: CSS

Uppgift

Den här laborationen handlar om att formatera en webbsida med CSS (Cascading Style Sheet). Vi kommer att utgå från det HTML-dokument som skapades i Laboration 1 och snygga till webbsidan så att den blir riktigt .

För godkänt ska din webbsida:

  • Länka ihop HTML-dokumentet från laboration 1 med ett CSS-stylesheet
  • Använda flera olika CSS-attribut, t.ex. font-familiy, font-size, line-height, text-align, margin, padding, border, color, height och width.
  • Använda CSS-selektorerna #id och .class
  • Använda element för att märka upp specifika delar av webbsidan t.ex. <div>, <span>, <header>, <section> och <footer>.
  • Validera i W3C:s CSS-validerare
  • Vara tillgänglig på http://www.student.itn.liu.se/~liuid###/

Redovisa för en labbassistent.

Var det för lätt?

Försök att utöka ditt stylesheet med följande styling

  • Positionera element med position.
  • Använd float för att ändra orienteringen av en <ul>.
  • Använd pseudoklasser (t.ex. tr:nth-child(odd)) för att färglägga varannan rad i ett <table>
  • Gör din layout responsiv med flexbox. Det vill säga, anpassa elementen på webbsidan efter skärmstorlek.
