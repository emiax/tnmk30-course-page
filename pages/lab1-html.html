TNMK30 Laboration 1: HTML

Uppgift

Denna laboration handlar om märkspråket HTML (Hypertext Markup Language). Labben går ut på att skapa en webbsida om något intressant ämne. Detta innebär att använda de tekniker som tagits upp under föreläsningen.

För godkänt ska din webbsida:

  • Använda följande html-taggar: <p>, <h1>, <a>, <table>, <tr>, <th>, <td>, <img>, <ul>, <li>, <strong>
  • Validera som HTML5 i W3C:s validerare
  • Vara tillgänglig på http://www.student.itn.liu.se/~liuid###/

    • Kodskelett för HTML5

    <!DOCTYPE html> <html> <head> <meta charset="utf-8"> <title>TNMK30 Lab 1</title> </head> <body> </body> </html>

    Texteditorer

    För att redigera html-dokument behövs en texteditor (ungefär som codeblocks).

    • Notepad++ - texteditor för Windows. Finns installerat labbsalen.
    • Sublime Text - texteditor för alla operativsystem.
    • TextMate - texteditor för (Mac) OS X.
    • Gedit - texteditor för Linux.

    Spara din webbsida

    Resultatet från denna laboration kommer att användas i labb 2. Då kommer vi att snygga till vår webbsida med CSS.
